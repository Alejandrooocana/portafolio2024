Electrónica Redes

- Resumen -

A lo largo de esta práctica, conoceremos, investigaremos e identificaremos distintos elementos que se utilizan para configurar una red local y el acceso a recursos de un equipo de red, para poder entender de manera práctica, como es que funcionan en conjunto estos elementos pertenecientes a las redes locales.

- Introducción -

Una red de área local o LAN es un grupo de computadoras y dispositivos periféricos que comparten una línea de comunicaciones común o un enlace inalámbrico a un servidor dentro de un área geográfica específica.

Una LAN permite a los usuarios conectarse a servidores internos, sitios web y otras LAN que pertenecen a la misma red de área amplia (WAN). Ethernet y Wi-Fi son las dos formas principales de habilitar las conexiones LAN. Ethernet es una especificación del Instituto de Ingenieros Eléctricos y Electrónicos (IEEE) que permite que las computadoras se comuniquen entre sí.

Este tipo de redes cuenta con elementos como: servidores, estaciones de trabajo, repetidores, puentes de red, hubs, switches, routers, etc. Estos elementos se ocuparán según sea la instalación que queramos realizar.

Existen muchas variaciones en cuanto a la configuración que se desee utilizar según el espacio y las necesidades del lugar donde se hacen este tipo de instalaciones, por lo que para poder darnos una idea de cómo funcionan distintos tipos de red, a continuación, se realizaran muchas variaciones en donde seremos capaces de identificar sus distintos elementos, así como sus descripciones y funcionamientos correspondientes.

Simulación

Diseño

Para el diseño de esta práctica, se utilizó el programa de Packet Tracer de Cisco para simular las redes de computadora especificadas, asignándoles a cada una de las computadoras las IP pertenecientes a la red 192.168.0.0. Posteriormente, utilizando la ventana de comandos y la función ping, se verificó la comunicación entre cada una de las computadoras.
