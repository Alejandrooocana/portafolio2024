Electrónica Redes

- Resumen -

A lo largo de esta práctica se conocerá, identificará y comprobará el funcionamiento de una red de área local utilizando switches y Routers.

- Introducción -

Una red de área local o LAN es un grupo de computadoras y dispositivos periféricos que comparten una línea de comunicaciones común o un enlace inalámbrico a un servidor dentro de un área geográfica específica.

Una LAN permite a los usuarios conectarse a servidores internos, sitios web y otras LAN que pertenecen a la misma red de área amplia (WAN). Ethernet y Wi-Fi son las dos formas principales de habilitar las conexiones LAN. Ethernet es una especificación del Instituto de Ingenieros Eléctricos y Electrónicos (IEEE) que permite que las computadoras se comuniquen entre sí.

Este tipo de redes cuenta con elementos como: servidores, estaciones de trabajo, repetidores, puentes de red, hubs, switches, routers, etc. Estos elementos se ocuparán según sea la instalación que queramos realizar.

Existen muchas variaciones en cuanto a la configuración que se desee utilizar según el espacio y las necesidades del lugar donde se hacen este tipo de instalaciones, por lo que para poder darnos una idea de cómo funcionan distintos tipos de red, a continuación, se realizaran muchas variaciones en donde seremos capaces de identificar sus distintos elementos, así como sus descripciones y funcionamientos correspondientes.

- Materiales -

  • 6 Computadoras
  • Switch
  • Router
  • Cable de red

- Desarrollo -

Diseño

Para el diseño de este experimento, primero se utilizo un switch para que con cables directos se conectaran 3 computadoras y pudieran visualizar la tanto la página mostrada por la computadora con servicios de servidor, como el envío y recepción de paquetes.

Después, se utilizó un router para que la computadora con servicios de servidor pudiera conectarse via alámbrica y que todos los demás dispositivos fueran capaces a conectarse y visualizar la tanto la página mostrada por la computadora con servicios de servidor, como el envío y recepción de paquetes.

Construcción y Simulación

Para la construcción de esta práctica, primero se identificó y caracterizó eñ funcionamienro del switch y el router.

Posteriormente, se elaboró una red, utilizando la asignación automática de IP y seleccionando el tipo de cable adecuado para conectar 3 computadoras al switch, incluida una con servicios de servidor.

Después se elaboró una red, utilizando la asignación automática de IP, seleccionando el tipo de cable adecuado y un router conectando 6 dispositivos inalámbricamente.
