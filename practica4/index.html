- Introducción -

En la actualidad podemos notar un crecimiento importante en cuanto a la accesibilidad de información gracias al internet y a las distintas páginas web que se encuentran en este espacio. Nosotros conocemos como página web a un documento electrónico accesible a través de internet que posee información organizada y presentada de forma visual.

Para poder realizar estas páginas web es necesario usar distintos lenguajes de programación según sean las necesidades del usuario, aunque los más conocidos hoy en día son HTML, CSS y Java Script, tanto por personas que apenas inician en el mundo de la programación web, como de las personas que llevan años en el desarrollo de este campo.

Este tipo de programación es de suma importancia para crearnos un espacio en este mundo tan grande de la web por lo que a lo largo de esta práctica se realizará una pagina web de lo mas sencilla, para ir conociendo los conocimientos básicos sobre este tema.
