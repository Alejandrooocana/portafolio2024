- Resumen -

A lo largo de esta práctica se buscó conocer, identificar y comprobar el funcionamiento de una página web con programación PHP. Servidor Local, un servidor en la nube y se enviaron los datos con el módulo ESP32.

- Introducción -

PHP es un lenguaje de código abierto muy popular especialmente en el desarrollo web y por cómo se puede incrustar en el HTML, permitiendo desarrollar webs y aplicaciones web dinámicas, favoreciendo la conexión entre el servidor y la interfaz del usuario.

Este es un lenguaje versátil sirve para conectar ciertos elementos de una página con determinados datos, permitiendo que el contenido cambie con las peticiones del usuario, Esta petición se manda a un servidor que, a su vez, contacta con una base de datos de donde extrae información, esta información vuelve a la página que varía (o no) en función de esta.

- Materiales -

  • Computadoras
  • Servidor local
  • Esp32
  • Protoboard
  • 4 botones
  • LED, resistencia de 220 Ohms
  • 4 resistencias de 1Khom
