- Resumen -

A lo largo de esta práctica se comenzó a experimentar con distintos estilos gracias a la incorporación del lenguaje de programación de css, dándoles un estilo diferente en cada división de la pagina web y escribiendo los estilos utilizados, para tener una referencia de cómo se pueden apreciar visualmente.

- Introducción -

En la actualidad podemos notar un crecimiento importante en cuanto a la accesibilidad de información gracias al internet y a las distintas páginas web que se encuentran en este espacio. Nosotros conocemos como página web a un documento electrónico accesible a través de internet que posee información organizada y presentada de forma visual.

Para poder realizar estas páginas web es necesario usar distintos lenguajes de programación según sean las necesidades del usuario, aunque los más conocidos hoy en día son HTML, CSS y Java Script, tanto por personas que apenas inician en el mundo de la programación web, como de las personas que llevan años en el desarrollo de este campo.

Una parte fundamental en este tipo de programación es la estética de la página web, ya que habla mucho de la confianza que transmite, de lo accesible que es y de la manera en la que quieres transmitir la información ingresada en esta página, por lo que a lo largo de esta práctica se realizaran distintos estilos para aplicar en secciones distintas de la página utilizándolo como guía para conocer como se pueden utilizar en nuestras futuras páginas web.

- Desarrollo -

Se realizó el código con base en los objetivos de la práctica

Simulación

Para esta práctica no fue necesario ninguna simulación.

Diseño

Para el diseño de esta página, se dividió en 10 secciones, donde cada una debía tener un título y un texto de muestra para aplicarle una tipografía, tamaño, color, alineación, fondo y contorno diferente, creando combinaciones completamente distintas para cada uno de los textos de ejemplo.

Construcción

Para el diseño de esta página se utilizó un archivo Index para insertar los 10 títulos y textos de muestra que se querían visualizar en cada sección de la página web.

Posteriormente, accedimos a la página: https://www.w3schools.com/css/default.asp, que se utilizó como guía para saber que comandos poner en el programa, para lograr distintos tipos de estilos en cada sección.

Una vez conociendo esta información, creamos un archivo de CSS el cual se utilizó para darle distintos tipos de estilos al texto.

En este caso el documento de CSS se llamó “styles”, y dentro de él se crearon distintos grupos de estilos, con selectores (o nombres) distintos, y con combinaciones de fuentes, tamaños, alineaciones, fondos y bordes, completamente distintos en cada uno de estos estilos, como se puede observar a continuación:

- Conclusiones -

Finalmente, podemos concluir que es muy importante realizar este tipo de prácticas para irnos familiarizando con la aplicación de distintos estilos existentes dentro del lenguaje de CSS, lo que nos abre las puertas para que con nuestra creatividad podamos generar páginas visualmente agradables que sean amigables con el usuario y que ayuden a expresar de la mejor manera la información que queramos transmitir con ella.
