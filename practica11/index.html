Muestrario de elementos input en HTML

Nombre del elemento Forma de escribirse en HTML Definición Ejemplo
Button <input type="button"> Un botón interactivo.
Checkbox <input type="checkbox"> Una casilla de verificación.
Color <input type="color"> Un control para seleccionar un color.
Date <input type="date"> Un control para seleccionar una fecha.
Datetime-local <input type="datetime-local"> Un control para seleccionar una fecha y hora local.
Email <input type="email"> Un control para ingresar una dirección de correo electrónico.
File <input type="file"> Un control para seleccionar un archivo.
Hidden <input type="hidden"> Un campo oculto que no se muestra en la interfaz.
Image <input type="image"> Una imagen como botón de envío de formulario.
Month <input type="month"> Un control para seleccionar un mes y año.
Number <input type="number"> Un control para ingresar un número.
Password <input type="password"> Un campo de contraseña.
Radio <input type="radio"> Un botón de selección única dentro de un grupo.
Range <input type="range"> Un control para seleccionar un valor de un rango.
Reset <input type="reset"> Un botón para restablecer el formulario a sus valores predeterminados.
Search <input type="search"> Un campo de búsqueda.
Submit <input type="submit"> Un botón de envío de formulario.
Tel <input type="tel"> Un control para ingresar un número de teléfono.
Text <input type="text"> Un campo de texto de una línea.
Time <input type="time"> Un control para seleccionar una hora.
URL <input type="url"> Un control para ingresar una URL.
Week <input type="week"> Un control para seleccionar una semana y año.
